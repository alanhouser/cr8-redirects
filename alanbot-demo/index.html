AlanBot – AI Demo

This page is a small, self-contained demo Alan built to show how he uses AI tools in his workflow.

What is this?

This demo shows a lightweight version of AlanBot — an AI agent Alan trained using his own writing voice, portfolio content, support workflows, and real examples of how he solves technical problems.

```

It isn’t a product. It’s simply a peek into how Alan uses AI to:

  • Simplify documentation and support workflows
  • Explain complex troubleshooting in a clear, friendly way
  • Package UI/UX reasoning and support logic into reusable formats

The real agent lives in Chatbase and can answer questions about Alan's support experience, design work, and debugging approach.

This is just the demo wrapper — not the agent itself.
```
